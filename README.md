# Student Result Management System

A Django-based web application for managing student academic records and automatically generating results based on subject marks.

## Overview

The Student Result Management System allows users to store student details and subject-wise marks. The system automatically calculates total marks, average marks, grades, and pass/fail status.

This project was developed using Python and Django to practice database management, CRUD operations, and business logic implementation.

## Features

* Add Student Records
* Store Subject-wise Marks
* Calculate Total Marks Automatically
* Generate Grades Automatically
* Determine Pass/Fail Status
* Manage Student Information
* Database Integration Using Django ORM

## Technologies Used

* Python
* Django
* SQLite

## Grade Criteria

| Average Marks | Grade |
| ------------- | ----- |
| 90 and above  | A+    |
| 75 - 89       | A     |
| 60 - 74       | B     |
| 35 - 59       | C     |
| Below 35      | F     |

## Pass Criteria

A student is considered Pass only if marks in all subjects are greater than or equal to 35.

Otherwise, the student is marked as Fail.
